# Inline AI Editing

Edit selected text directly in your notes using AI — no need to copy/paste into a chat.

---

## How to Use

1. **Select text** in any note
2. **Trigger** via:
   - Right-click &rarr; **Edit with AI** (requires "Native menus" disabled in Obsidian settings)
   - Command palette &rarr; **Edit selection with AI**
3. **Enter instruction** (e.g., "Make this more concise", "Translate to French", "Fix grammar")
4. **Generate** — Watch the streaming preview with cancel option
5. **Review & Apply** — Edit the result, retry with a new instruction, or apply to replace the original text

---

## Capabilities

| Feature | Description |
|---------|-------------|
| Model Selection | Dropdown to choose from any configured provider |
| Streaming Preview | See AI output as it generates |
| Cancel Support | Stop generation mid-stream |
| Editable Result | Modify the AI output before applying |
| Retry | Regenerate with the same or a new instruction |
| Undo | Standard Ctrl/Cmd+Z after applying |
