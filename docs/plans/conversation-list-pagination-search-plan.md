#### Design Approach

**Pagination**: Offset-based (page/pageSize) using existing LIMIT/OFFSET infrastructure. "Load More" button (not infinite scroll) — sidebar is narrow, scroll events imprecise, button gives explicit user intent.

**Search**: FTS5 title search replaces the paginated list temporarily. Clearing search returns to paginated browse at page 0. Search results also support pagination for consistency.

**State**: ConversationManager accumulates pages (page 0 + page 1 + ...) and owns all pagination/search state. ConversationList stays a pure renderer.

#### Key Decisions

| Decision | Options | Recommendation | Rationale |
|----------|---------|----------------|-----------|
| Load More vs Infinite Scroll | Button / IntersectionObserver | **Button** | Simpler, predictable, sidebar-friendly, accessible |
| Offset vs Cursor pagination | Offset / Cursor | **Offset** | Matches existing PaginatedResult infrastructure |
| Search scope (MVP) | Title FTS / Title + Content | **Title FTS only** | Already wired, <10ms latency. Semantic search is Phase 2 |
| Search + pagination interaction | Replace list / Filter in-place | **Replace list** | Cleaner UX — search is a different mode |
| hasMore signal | Explicit from backend / Infer from length | **Explicit** | PaginatedResult already provides hasNextPage |

#### Interface Contracts

**ConversationManager (UI) — new API:**
```typescript
// State
hasMore: boolean
isLoading: boolean
isSearchActive: boolean

// Methods
loadConversations(): Promise<void>       // Initial load (page 0, resets state)
loadMoreConversations(): Promise<void>   // Next page, append to existing
searchConversations(query: string): Promise<void>  // FTS search, replace list
clearSearch(): Promise<void>             // Return to paginated browse at page 0
```

**ConversationList (UI) — new callbacks:**
```typescript
onLoadMore: () => void
onSearch: (query: string) => void
onSearchClear: () => void
```

**ChatService — updated signature:**
```typescript
listConversations(options?: {
  limit?: number;
  page?: number;
  sortBy?: string;
  sortOrder?: 'asc' | 'desc';
}): Promise<PaginatedResult<ConversationListItem>>
```
